How to decompress .zs files

Simply opening .zs files will not always work, as they have to be decompressed first for some software. However, Switch-Toolbox should be able to open them (if not, check if you have the newest version).

Preparations

For this you will need the Totk.ZStdTool. Download zstd-tool.exe from the newest release.

Totk.ZStdTool relies on the .NET 7 Runtime. If you do not have it, opening Totk.ZStdTool will not do anything. You can download it here from Microsoft (for most people, choose x64 installer).

Decompression

IMPORTANT: You will first have to provide a Game Path. Open zStdTool and click Settings. Then, simply copy paste the path to your dumped game from explorer, namely the path to the romfs folder.

You can either decompress single files or whole folders. First, click Browse for either file or folder and choose the one you want to decompress. Then press Decompress (Folder). Next, you will be prompted to select a destination. After selecting the destination, the decompression will start automatically. The file(s) will be copied and decompressed to the location you chose. The original file(s) will not be changed.

Important Considerations

Simply decompressing the 3d model files in the /model folder will NOT be enough for viewing/editing/exporting them with Switch-Toolbox or other tools. You will need to decompress the 3d models with a separate tool.

If you get any errors on your first try, double check if you have set your Game Path correctly.
